Durbar Square in Kathmandu, Nepal before the earthquake in Nepal on April 25, 2015.

The ‘before’ model was generated using SURE, photogrammetric software (nframes.com), from high altitude aerial imagery. The imagery was acquired using the MACS camera system by DLR Berlin (dlr.de/macs) prior to the earthquake.

The ‘after’ model was created using UAS (unmanned aerial system) and photogrammetry by the company Drones Imaging (http://www.dronesimaging.com/en).
